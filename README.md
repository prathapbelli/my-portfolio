# Netflix Style Portfolio

This is a portfolio project that mimics the style of Netflix, built with React and TypeScript.

## Features

- Netflix-inspired UI
- Project cards and modals displaying project details
- Skills, experience, and personal info sections
- Responsive and modern design

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/prathapbell/netflix-style-portfolio.git
cd netflix-style-portfolio
```

### 2. Install Dependencies

Assuming you are using [npm](https://www.npmjs.com/):

```bash
npm install
```

### 3. Run Locally

To start the development server:

```bash
npm run dev
```

Open your browser and go to [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Customizing

- Update your personal details, projects, and skills in the `constants.ts` file.
- Add or modify images in the `public` directory or use external image URLs.
- Edit or add new components as required for your needs.

## Deploy

You can deploy this portfolio easily using free online platforms.
